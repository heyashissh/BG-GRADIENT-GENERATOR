# Gradient Background Generator

A simple and interactive web application that generates beautiful random gradient backgrounds using JavaScript. Users can create unique color combinations with a button click and instantly copy the generated CSS code for use in their own projects.

## Features

- Generate random gradient color combinations
- Real-time background updates
- Copy CSS gradient code to clipboard
- Responsive and user-friendly interface
- Built with Vanilla JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

- JavaScript Functions
- CSS Linear Gradients
- Clipboard API
